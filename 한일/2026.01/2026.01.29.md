4. [경기2 고객센터 특별교육 가스링크 보완요청사항]
    1. 검침파트 단체문자 보낼 때 화면 튕김 현상 개선 요청
      - 개선 가능
    2. 계량기 자동인식 10등급 이상도 되도록 요청
      - 개선 가능
    3. 납기 시작 시 가스링크 튕김 현상 발생 개선 요청
      - 서버 요청이 너무 많아서 생기는 현상임
      - 서버쪽에 1차적으로 보완 처리를 해둠 (1/27)
      - 혹시나 추가 발생시
        - 처리방안 1 : 티켓팅 방식으로 처리 할까 고려하고 있음
        - 처리방안 2 : 네트워크 또는 서버 증설 (비용문제가 있음)
    4. 사전문자 보낼 경우 과거 시간이 선택되어 보내지는 경우가 있음(최소 현재보다 과거가 선택되지 않도록 개선)
        - 개선 가능
    5. 문자 직접입력하여 보내는 경우 기존 기간계 전송에서 가스링크 통해서 나가도록 수정
        - 개선 가능
    6. 안전점검 시 취사전용 세대 보일러 항목 건너띄기 되는 부분 건너띄기 되지 않도록 수정(CO경보기 설치 유무 체크를 위함), 취사전용은 CO경보기 디폴트값을 '미설치'로 해줄 수 있는지 확인 요청
        - 건너띄지 않도록 가능
        - 취사전용시 미설치 디폴트 가능

5. 점검할때 부적합발생한 개선거 당일날 개선권고가면 당일처리가안됨 (완료)
    - 화면 : /safety-check/improvement-advice
    - 개선처리 버튼 조건 : 
        1. 현재 사진 촬영 모드가 아니고

        3. 아직 개선되지 않았고
        4. 오늘 방문등록 이력이 없을 때만
    - 작업 내용 : 2번 조건 제거
