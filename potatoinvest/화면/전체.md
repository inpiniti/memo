# 기본 레이아웃

<header>	: 페이지 최상단
<nav>		: Hyperlink, 메뉴바
<aside>		: 카테고리, 현재 페이지 외의 컨텐츠
<main>		: 현재 페이지 주 컨텐츠가 포함
<section>	: main 내 컨텐츠를 구역별로 다룸
<footer>	: 페이지 최하단
